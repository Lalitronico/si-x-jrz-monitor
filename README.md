# Monitor de Brigadas — Sí X Juárez

Dashboard de monitoreo semanal de encuestas por distrito.

## Actualización semanal

1. Exportar CSVs de cada distrito desde Alchemer
2. Actualizar el bloque `DATA` en `index.html` con los nuevos datos
3. Push al repo → GitHub Pages se actualiza automáticamente

## Publicación

Publicado con GitHub Pages desde la rama `main`.
